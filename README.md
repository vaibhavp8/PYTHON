# Python Fundamentals Mini Projects

> Learning from the basics to AI Engineering and Machine Learning

Welcome to **Python Fundamentals Mini Projects**! This repository is a comprehensive collection of mini projects and exercises designed to help you master Python programming, from foundational concepts through advanced topics in AI Engineering and Machine Learning.

## 📚 Overview

This repository serves as a learning resource for:
- **Python Basics**: Variables, data types, control flow, functions
- **Intermediate Concepts**: OOP, file handling, APIs, data structures
- **Advanced Topics**: Data Analysis, Machine Learning, AI Engineering
- **Practical Applications**: Real-world projects and use cases

## 🎯 Learning Path

Whether you're a complete beginner or looking to level up your skills, this repository provides structured mini projects organized by difficulty and topic.

### Beginner Level
- Python fundamentals and syntax
- Data types and operations
- Control structures (loops, conditionals)
- Functions and modules

### Intermediate Level
- Object-Oriented Programming (OOP)
- File I/O and data handling
- Working with APIs
- Error handling and debugging

### Advanced Level
- Data Analysis with Pandas and NumPy
- Data Visualization
- Machine Learning basics
- AI Engineering concepts

## 🚀 Getting Started

### Prerequisites
- Python 3.8 or higher
- pip (Python package manager)
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/vaibhavp8/python-fundamentals-mini-projects.git
   cd python-fundamentals-mini-projects
   ```

2. **Create a virtual environment** (recommended)
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

## 💻 How to Use

Each project folder contains:
- **`main.py`** or **`solution.py`** - The implementation
- **`README.md`** - Project description and objectives
- **`test.py`** (optional) - Unit tests to verify your solution
- **`requirements.txt`** (optional) - Project-specific dependencies

### Running a Project

```bash
cd beginner/01-basics
python main.py
```
